# Projet_7_OPEN_CLASSROOMS



<h1> GROUPOMANIA </h1>

<p> Ce site est un réseau social pour les employés de Groupomania, où ils peuvent échanger à travers des posts.</p>


<h2>Installation</h2>

<ol> Front-end:
  <li>cd frontend</li>
  <li>npm install</li>
  <li> npm start </li>
</ol>

<ol> Back-end:
   
   <li>cd backend</li>
   <li>npm install</li>
   <li>npx sequelize cli db:migrate(pour récupérer les données)</li>
   <li>npm start</li>
</ol>
